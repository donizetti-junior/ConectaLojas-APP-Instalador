Você está no iPhone/iPad?
Esta versão é só para Android. A versão iOS ainda não está disponível.
Como instalar no seu Android
1
Toque em Baixar para Android acima.
2
Se o Android avisar que o arquivo pode prejudicar o dispositivo, escolha Manter mesmo assim.
3
Abra o arquivo baixado e toque em Instalar. Se pedir permissão pra instalar de fontes desconhecidas, ative pro Chrome (ou o navegador que estiver usando).
4
Pronto! Abra o app ConectaLojas no menu.
Versão de teste. Este app está em desenvolvimento. Se encontrar algum problema, avise!
